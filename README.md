# RT STORYLOADING

A modern and clean loading screen featuring background character images, music control, and easy customization.

---

## 🔍 Preview
Click the link below to see the loading screen in action:

👉 **[Watch Showcase](https://youtu.be/xcxLZHVxYYI)**  

---

## 🎨 Features
- Background with **character images**
- Fully customizable **background visuals**
- Built-in **music player**
- Adjustable **music volume**
- Clean and modern UI
- Easy configuration

---

## ⚙️ Customization
You can easily customize the loading screen in the `config.js` file:

- Change **background images (characters)**
- Replace or edit **music**
- Adjust **volume settings**
- Modify texts and sections

Make it fit perfectly to your server style.

## 📦 Installation
1. Download or clone this repository  
2. Add the resource to your `resources` folder  
3. Configure everything in `config.js`  
4. Start/restart your server
